# 김상훈의 SKALA 프론트엔드 과제

HTML로 기본 페이지를 만든 뒤 CSS로 꾸미고, JavaScript로 버튼과 날씨 기능을 추가했습니다. 예제 내용은 제 관심사와 이번 주 수업 일정에 맞게 바꿨습니다.

## 접속하기

아래 링크에서 결과를 확인할 수 있습니다.

### [웹사이트 바로 보기](https://agenticlab-sh.github.io/skala-front/)

로컬에서 확인할 때는 다음 순서로 실행합니다.

1. Live Server를 실행합니다.
2. 이 폴더의 `index.html`을 엽니다.
3. 상단 메뉴에서 내 소개, 시간표, 휴일, 여행지, 회원가입 페이지로 이동합니다.

## 페이지 구성

| 페이지 | 내용 |
|---|---|
| `html/index.html` | 전체 메뉴, JavaScript 과제, 실시간 날씨 |
| `html/myProfile.html` | 좋아하는 음식, 올해 목표, 나를 설명하는 단어 |
| `html/myClass.html` | 실제 이번 주 수업 일정과 셀 병합 표 |
| `html/holiday.html` | 시간대별 대체휴일 루틴 |
| `html/myTrip.html` | 서울 야경과 겨울 산을 소개하고 오디오·비디오를 넣은 여행지 페이지 |
| `html/signUp.html` | GET 방식 입력 폼 |
| `html/signUpResult.html` | 전달된 입력값을 읽어 보여주는 결과 화면 |
| `html/mediaSources.html` | 사용한 미디어의 원본과 라이선스 |

## 사용한 내용

- HTML: 목록, 설명 목록, 표, 링크, 폼, 오디오, 이미지, 비디오, 시맨틱 태그
- CSS: Google Font, 공통 색상, 박스 모델, Flexbox, Grid, 반응형 화면
- JavaScript: 배열, 객체, 반복문, 조건문, `prompt`, `alert`, DOM 이벤트, `fetch`, 모듈 분리

## 추가로 해본 것

- 버튼을 누르면 밝은 화면과 어두운 화면이 바뀌도록 했습니다.
- 메인 화면에 현재 시각을 표시했습니다.
- 회원가입 화면에 비밀번호 보기와 자기소개 글자 수 표시를 넣었습니다.

HTML 페이지에 공통 `style.css`를 연결하고, 화면 크기에 따라 배치가 달라지도록 구성했습니다. 회원가입 폼은 입력 연습용이며 실제 회원가입이나 정보 저장 기능은 없습니다. 결과 화면에는 비밀번호가 나타나지 않습니다.

시간표에는 이번 주 과목을 날짜별로 표시했습니다. 점심시간 행에는 표의 셀 병합을 적용했습니다.

## 만들면서 확인한 점

- `html` 폴더에서 CSS와 미디어를 연결할 때는 `../`로 상위 폴더를 먼저 찾아가야 했습니다.
- GET 방식으로 보낸 값은 주소에 붙어서 전달되고, 결과 페이지에서 `URLSearchParams`로 읽을 수 있었습니다.
- 날씨 API를 부르는 코드와 화면을 바꾸는 코드를 두 파일로 나누고 `export`, `import`로 연결했습니다.

사진·음악·영상의 출처와 사용 조건은 [`html/mediaSources.html`](./html/mediaSources.html)에 정리했습니다.
